## Agenda 和准备材料

- 讨论的形式和话题话题枚举 
  - 要录屏吗？录屏 
  - 讨论维度是什么？ 
    - 不同项目的工作中方案？不同的项目都是怎么做的？ 有哪些坑？ 做一些案例输入
    - 方案优点，亮点 
    - 方案缺点，局限性 
    - 其他可选的代替方案 
    - 推荐主流的方式 
    - 有没有参考案例资料
    - 产出是什么？整理成会议纪要，更像是一种方案盘点
    - 原则：以严肃对待、生产可用、深入为原则。避免 "这个很简单，但是项目总是会发生问题"。
- 本次讨论的议题，例如："业务单号生成实现的最佳方案"
